## Correr en dev

1. Clonar el repositorio
2. Instalar dependencias ```npm install```
3. Correr las migraciones de Prisma ```npx prisma migrate dev```
4. Correr el proyecto 
```bash
npm run dev
# or
npm run dev -- -H 0.0.0.0  # para movil y web
# or
yarn dev
# or
pnpm dev
# or
bun dev
```
